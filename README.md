# Cowsay
  This is a simple script in Python3 named Cowsay; the script is a cow that says what you give it as argument.
# Usage
``` shell
  $> python3 cawsay.py <text>
```
  #### Example:</br>
``` shell
  $> python3 cawsay.py 1337 Future Is Loading...
```
